Site Overview

Welcome to College Scorecard Data. This site was created as a school project to demonstrate the Model-View-Class (MVC) framework with Entity Framework Code-First and RESTful API consumption. We hope you find the UI easy to use. All data are sourced from Data.gov

Explore the Data

Our Data files contain Institution-level data files for 1996-97 through 2018-19 containing aggregate data for each institution. Includes information on institutional characteristics, enrollment, student aid, costs, and student outcomes.
